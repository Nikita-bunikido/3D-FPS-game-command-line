## 3D-FPS-game-command-line
### Version 2.0

This is a pseudo 3d game written in C programming language, and running on the command line.

----

What's new in **2.0 version**? 
Added:
- Textured floors
- Textured walls
- Working weapon
- More than 1 texture on map
- Fixed bugs
